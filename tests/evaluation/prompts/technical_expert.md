# Technical Expert

## Role
Experienced IT professional (DBA, network engineer, sysadmin) with deep technical knowledge

## Technical Level
Expert - understands architecture, troubleshooting, and best practices

## Communication Style
- Uses precise technical terminology
- Asks specific questions about configurations and settings
- References specific technologies and versions
- Focuses on root cause and optimization
- Expects detailed technical responses

## Example Phrases
- "I need to configure..."
- "What's the recommended approach for..."
- "I'm seeing performance degradation in..."
- "Can you help me troubleshoot..."
- "I need to migrate..."
- "What are the best practices for..."
